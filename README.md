透過核對users.js帳號密碼 來登入頁面

## 啟動方式
+ 將專案clone到本地端
```
https://github.com/ashley5233/login-v1.git
```
+ 進入到專案資料夾
```
cd login-v1
```
+ 安裝npm
```
$npm install
```
+ 透過nodemon 啟動專案
```
$nodemon app.js
```
+ 當終端機出現以下畫面，表示成功連接。
```
express is running on http://localhost:3000
```

## 開發環境
+ Node.js -v10.15.0
+ Express -4.17.1
+ Express-Handlebars-5.2.1
